# AGL Essentials

**AGL Essentials** is a Visual Studio Code extension designed to simplify and enhance AGL (Accenture Gateway Layer) development within the **Opus TV system**. It provides essential tools such as an **Endpoint Viewer** and a **Mapper Viewer** to boost productivity and streamline the development process.

### 2. Mapper Viewer
- Visualize and modify data mappings in the AGL layer.
- Handle complex mapping scenarios with ease.
- Highlight and validate errors in mapping configurations.

## ğŸ“ Usage

### Open the Endpoint Viewer
1. Press `Ctrl+Shift+P` to open the Command Palette.
